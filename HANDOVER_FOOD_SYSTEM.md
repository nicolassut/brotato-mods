## 4. THE SPEC to build (from the design doc + session laws — verbatim rules)

### Buff engine ("shared timer with diminishing extensions")
- Each food type has ONE buff instance per player: magnitude stacks up, timer is shared.
- Stack 1: full base duration. Stack 2 adds +50% of base. Stack 3 adds +25%. Stack 4+ adds
  +25% (floor). All percentage-based: anything modifying buff duration scales base AND
  extensions proportionally. Fractional seconds round down.
- Appetite scaling doctrine: food buff strength = base + (ratio × Appetite). Damage-dealing
  food effects ALSO scale with their native damage stat (dual scaling). Always-on item stat
  lines stay flat. Appetite duration bonus: +10% base duration per 10 Appetite (percent, not
  flat).
- Foods COUNT as consumables (so consumable-triggered vanilla content reacts to them), but
  spawn ONLY via their spawner item (no default drops).
- HUD: a visible buff indicator is desirable but optional this pass (flag it).
